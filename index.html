📋 サイト説明

ファビアチェックは、ファビオ（恐怖症）をお持ちの方が安心して映画を楽しめるよう、事前に恐怖要素を確認できる情報サイトです。

🎯 主な機能

  • 恐怖要素チェック：20種類の恐怖要素を事前に確認
  • 映画検索：タイトルで映画を検索
  • カテゴリ別表示：公開予定、上映中、話題、評価の高い映画
  • ユーザー投稿：皆様で情報を共有・更新

🏷️ 恐怖要素一覧

高所、閉所、暗所、蜘蛛、蝶、蟻、昆虫、蛙、ヘビ、集合体、血液描写、水・溺水、雷、嘔吐、注射・先端、ピエロ、幽霊・超常現象、性的シーン、死体、その他

🤝 ご協力のお願い

映画カードの編集ボタン（✏️）から、恐怖要素情報を追加・修正していただけます。皆様のご協力により、より多くの方が安心して映画を楽しめるようになります。

📊 データソース

映画情報は The Movie Database (TMDB) API を使用しています。

⚠️ 免責事項

🎬 映画情報について

本サイトで提供される映画の恐怖要素情報は、ユーザーの皆様からの投稿に基づいています。情報の正確性については完全に保証できませんので、ご了承ください。

📝 個人差について

恐怖症の感じ方には個人差があります。同じシーンでも人によって感じ方が異なる場合がありますので、最終的な判断はご自身でお願いいたします。

🔄 情報の更新

映画情報は随時更新されますが、最新の情報については必ず公式サイト等でご確認ください。

📱 技術的制限

本サイトはTMDB APIを使用しており、API側の制限により一部の機能が制限される場合があります。

💾 データの取り扱い

投稿された情報は、サイトの改善と他のユーザーへの情報提供のために使用させていただきます。個人を特定できる情報の投稿はお控えください。

📞 お問い合わせ

本サイトに関するお問い合わせや情報の修正依頼がございましたら、適切な方法でご連絡ください。
